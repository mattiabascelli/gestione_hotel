Gestione Hotel Alpha

Sembra un sito ma in realtà..

GESTIONE HOTEL è una web application sviluppata dal sottoscritto Mattia Bascelli durante il corso di Java Developer di BitCamp

Presso la pagina Github dedicata potrete scaricare tutto il materiale riguardante il programma:

  • Diagramma ER
  • Diagramma UML
  • Codice JAVA.zip

Per funzionare il programma utilizza le seguenti librerie JAVA:

  • mysql-connector-j-8.2.0 [download]
  • pdfbox-app-2.0.30 [download]

Inoltre per il funzionamento dell'applicazione bisogna installare la JVM scaricabile qui

Un breve video dimostrativo dell'applicazione.

Il programma genera automaticamente il database con cui su MySQL.
L'interfaccia web permetta all'utente finale di avere un'accesso UserFriendly al gestionale.
